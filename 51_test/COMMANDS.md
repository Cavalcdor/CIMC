# 舵机调试工具 — 串口指令手册

## 📡 连接参数

| 参数 | 值 |
|------|-----|
| 波特率 | **115200** |
| 数据位 | 8 |
| 停止位 | 1 |
| 校验位 | 无 |

> 上电后发送任意字符，MCU 会回复 `>` 提示符，表示就绪。

---

## 📋 指令速查

| 指令 | 模式 | 功能 |
|------|------|------|
| `0` ~ `7` | — | 选中单个舵机 |
| `f0` ~ `f3` | — | 选中手指对 |
| `+` | 单舵机 | PWM **增加**步长 |
| `+` | 手指对 | 手指**弯曲** |
| `-` | 单舵机 | PWM **减少**步长 |
| `-` | 手指对 | 手指**伸直** |
| `s` | 全局 | 打印全部 8 个舵机当前 PWM |
| `c` | 全局 | **回中**：全部设为 300 |
| `n` | 全局 | **切换步长**：1 → 5 → 10 → 20 → 50 → 循环 |
| `a` | 手指对 | **自动扫角**：从 200 到 400 逐步扫描 |
| `p` | 全局 | 显示当前选中的舵机/手指信息 |
| `h` | 全局 | 打印帮助列表 |

---

## 🖐 舵机与手指对应关系

| 手指 | 命令 | 舵机通道 | 弯曲方向 |
|------|------|---------|---------|
| **拇指** (Thumb) | `f0` | S0, S1 | S0 ↑ , S1 ↓ |
| **食指** (Index) | `f1` | S2, S3 | S2 ↑ , S3 ↓ |
| **中指** (Middle) | `f2` | S4, S5 | S4 ↑ , S5 ↓ |
| **小指** (Pinky) | `f3` | S6, S7 | S6 ↑ , S7 ↓ |

> 每根手指由 **2 个舵机拮抗驱动**：一个拉紧时弯曲，另一个拉紧时伸直。

## 🔰 典型调试流程

### 目标：找到每根手指的「弯曲限位」和「伸直限位」PWM 值

#### Step 1 — 选中手指对

```
发 →  f0
回 ←  [F0] Thumb: S0=300 S1=300  (+bend/-straighten)
```

#### Step 2 — 微调找到限位

反复按 `+` 观察手指弯曲，直到**不能再弯但舵机不堵转**的位置：

```
发 →  +
回 ←  F0: S0=310 S1=290
发 →  +
回 ←  F0: S0=320 S1=280
... 继续直到弯曲极限 ...
发 →  +
回 ←  F0: S0=400 S1=200   ← 记下这个值！
```

然后按 `-` 观察手指伸直，找到伸直限位：

```
发 →  -
回 ←  F0: S0=390 S1=210
... 继续直到伸直极限 ...
发 →  -
回 ←  F0: S0=220 S1=380   ← 记下这个值！
```

#### Step 3 — 记录结果

把记下的值填入下表，后续写入 `51/51.c`：

```
拇指弯曲位: S0=___ S1=___    拇指伸直位: S0=___ S1=___
食指弯曲位: S2=___ S3=___    食指伸直位: S2=___ S3=___
中指弯曲位: S4=___ S5=___    中指伸直位: S4=___ S5=___
小指弯曲位: S6=___ S7=___    小指伸直位: S6=___ S7=___
```

#### Step 4 — 切换到下一个手指

```
发 →  f1     ← 测试食指
发 →  f2     ← 测试中指
发 →  f3     ← 测试小指
```

## 🔧 进阶技巧

### 调整步长

默认步长为 **10**。需要精细微调时可以切换步长：

```
发 →  n      ← 步长切换为 5
回 ←  Step= 5
发 →  n      ← 步长切换为 1（最精细）
回 ←  Step= 1
发 →  n      ← 步长切换回 50（快速移动）
回 ←  Step=50
```

步长循环顺序：**1 → 5 → 10 → 20 → 50 → 1 → ...**

### 查看全局状态

随时查看所有舵机的当前 PWM：

```
发 →  s
回 ←  S0:300  S1:300
     S2:300  S3:300
     S4:300  S5:300
     S6:300  S7:300
     >
```

### 回中复位

手指调乱了，一键全部回中：

```
发 →  c
回 ←  All centered (300)
```

### 自动扫角

让当前手指对自动从 200 扫到 400，观察全程运动：

```
发 →  f1     ← 选中食指
发 →  a      ← 自动扫角
```

手指会：
1. 回中 → 保持 1s
2. 逐步弯曲（S2↑ S3↓）→ 保持
3. 逐步回到中位 → 继续伸直（S2↓ S3↑）→ 保持
4. 逐步回中

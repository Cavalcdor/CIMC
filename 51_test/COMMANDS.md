# 舵机调试工具 — 串口指令手册

## 📡 连接参数

| 参数 | 值 |
|------|-----|
| 波特率 | **115200** |
| 数据位 | 8 |
| 停止位 | 1 |
| 校验位 | 无 |
| 晶振 | 11.0592MHz |
| TH1/TL1 | 0xFD |

> **上电后**：MCU 会连续发送 3 次欢迎信息，确保串口助手无论何时打开都能收到。

---

## 📋 指令速查

| 指令 | 功能（含义始终一致） |
|------|---------------------|
| `0` ~ `7` | **选中舵机 N**，进入单舵机模式 |
| `f0` ~ `f3` | **选中手指对**（0=拇指 1=食指 2=中指 3=小指），进入耦合模式 |
| `i` 或 `I` | **个体模式**：进入/切换手指对内的某个舵机（进入后 `i` 切换 0↔1） |
| `b` 或 `B` | **返回耦合**：从个体模式回到耦合模式 |
| `+` | 步进+（单舵机/个体=增加PWM，耦合=弯曲） |
| `-` | 步进-（单舵机/个体=减少PWM，耦合=伸直） |
| `s` | 打印全部 8 个舵机当前 PWM |
| `Sxx` | **直接设置步长**，如 `S25` → step=25 |
| `c` 或 `C` | **回中**：所有舵机设为 300 |
| `n` 或 `N` | **切换步长**：10 → 20 → 50 → 1 → 5 → 循环 |
| `a` 或 `A` | **自动扫角**：当前手指对从弯曲到伸直完整扫一遍 |
| `p` 或 `P` | 显示当前选中的舵机/手指信息 |
| `h` 或 `H` | 打印帮助列表 |
| 其他字符 | 回复 `?` 表示未知命令 |

---

## 🖐 舵机与手指对应关系

| 手指 | 命令 | 舵机通道 | 弯曲方向 |
|------|------|---------|---------|
| **拇指** (Thumb) | `f0` | S0, S1 | S0 ↑ , S1 ↓ |
| **食指** (Index) | `f1` | S2, S3 | S2 ↑ , S3 ↓ |
| **中指** (Middle) | `f2` | S4, S5 | S4 ↑ , S5 ↓ |
| **小指** (Pinky) | `f3` | S6, S7 | S6 ↑ , S7 ↓ |

> 每根手指由 **2 个舵机拮抗驱动**：一个拉紧时弯曲，另一个拉紧时伸直。
>
> **弯曲**：前一舵机 ↑（增大），后一舵机 ↓（减小）
> **伸直**：前一舵机 ↓（减小），后一舵机 ↑（增大）

## 🔰 典型调试流程

### 目标：找到每根手指的「弯曲限位」和「伸直限位」PWM 值

#### Step 1 — 选中手指对

```
发 →  f0
回 ←  [F0] Coupled: S0=300 S1=300  (+bend/-straighten)
>
```

#### Step 2 — 微调找到弯曲限位

反复按 `+` 观察手指弯曲，直到**不能再弯但舵机不堵转**的位置：

```
发 →  +
回 ←  F0: S0=310 S1=290
>
发 →  +
回 ←  F0: S0=320 S1=280
>
... 继续 ...
发 →  +
回 ←  F0: S0=400 S1=200
>
```

⛔ 到了物理极限还继续按 `+`，会回复 `limit`：

```
发 →  +
回 ←  limit
>
```

👉 **记下：** **`S0=___ S1=___`（弯曲位）**

#### Step 3 — 微调找到伸直限位

按 `-` 观察手指伸直：

```
发 →  -
回 ←  F0: S0=390 S1=210
>
... 继续 ...
发 →  -
回 ←  F0: S0=220 S1=380
>
```

👉 **记下：** **`S0=___ S1=___`（伸直位）**

#### Step 4 — 记录这组数据

```
拇指弯曲位: S0=___ S1=___    拇指伸直位: S0=___ S1=___
食指弯曲位: S2=___ S3=___    食指伸直位: S2=___ S3=___
中指弯曲位: S4=___ S5=___    中指伸直位: S4=___ S5=___
小指弯曲位: S6=___ S7=___    小指伸直位: S6=___ S7=___
```

#### Step 5 — 切换到下一个手指

```
发 →  f1     ← 测试食指
发 →  f2     ← 测试中指
发 →  f3     ← 测试小指
```

## 🔧 进阶技巧

### 调整步长

默认步长为 **10**，按 `n` 切换：

```
发 →  n
回 ←  Step=20
>
发 →  n
回 ←  Step=50
>
发 →  n
回 ←  Step= 1    ← 最精细（逐个微调）
>
```

步长循环：**10 → 20 → 50 → 1 → 5 → 10 → ...**

### 手指对内个体舵机控制

选中手指对后，用 `i` 进入个体模式单独调整某个舵机；`i` 切换 S0/S1；`b` 返回耦合：

```
发 →  f0                ← 进入手指对模式（耦合）
回 ←  [F0] Coupled: S0=300 S1=300  (+bend/-straighten)
>
发 →  i                 ← 进入个体模式，选中 S0
回 ←  [F0.S0] PWM=300
>
发 →  +                 ← S0 单独增加
回 ←  F0.S0=310
>
发 →  +                 ← S0=320
发 →  i                 ← 切换到 S1
回 ←  [F0.S1] PWM=300
>
发 →  +                 ← S1 单独增加
回 ←  F0.S1=310
>
发 →  b                 ← 返回耦合模式
回 ←  [F0] Coupled: S0=320 S1=310  (+bend/-straighten)
>
```

> 个体模式下 `+`/`-` 仅调整当前选中的那个舵机，不影响对舵机。
> `c`（回中）在任何模式下含义一致，始终将所有舵机置 300。
> `0`-`7` 在任何模式下都切换到单舵机模式。

### 直接设置步长

除了用 `n` 循环切换，还可以用 `Sxx` 直接指定步长值：

```
发 →  S25               ← 步长设为 25
回 ←  Step=25
>
发 →  S03               ← 步长设为 3（注意补零成两位）
回 ←  Step=3
>
```

> `S` 后面必须跟 **恰好 2 位数字**，如 `S05`(步长5)、`S15`(步长15)、`S50`(步长50)

### 单舵机单独控制

需要单独控制某个舵机时：

```
发 →  0       ← 选中舵机 0
回 ←  [S0] PWM=300
>
发 →  +       ← S0=310
发 →  +       ← S0=320
发 →  -       ← S0=310
>
发 →  s       ← 查看全局
回 ←  S0:310  S1:300
     S2:300  S3:300
     S4:300  S5:300
     S6:300  S7:300
>
```

### 查看全局状态

```
发 →  s
回 ←  S0:300  S1:300
     S2:300  S3:300
     S4:300  S5:300
     S6:300  S7:300
>
```

### 查看当前选中

```
发 →  p
回 ←  [S0] PWM=300    ← 当前选中的信息
>
```

### 回中复位

手指调乱了，一键全部回中：

```
发 →  c
回 ←  All centered (300)
>
```

### 自动扫角

让当前手指对完整扫一遍，观察全程运动：

```
发 →  f1     ← 选中食指
发 →  a      ← 自动扫角
```

手指自动完成 4 个阶段（无需人工干预）：
1. **回中** → 保持 1s
2. **逐步弯曲** (S2↑ S3↓) → 保持 0.8s
3. **逐步回到中心 → 继续伸直** (S2↓ S3↑) → 保持 0.8s
4. **逐步回中** → 打印 `F1 sweep done`

## 💡 推荐测试顺序

```
1. 烧录程序 → 打开串口助手 (115200)
2. 看到 3 次欢迎信息 → 发 h 确认通信正常
3. 先测拇指 (f0) → 找到弯曲/伸直限位 → 记下
4. 再测食指 (f1) → 找到弯曲/伸直限位 → 记下
5. 再测中指 (f2) → 找到弯曲/伸直限位 → 记下
6. 最后测小指 (f3) → 找到弯曲/伸直限位 → 记下
7. 把 4 组数据填入 51/51.c 的手势定义中
```

## 📝 记录表模板

**舵机 — 手指对应关系：**

```
S0, S1 = 拇指     S2, S3 = 食指
S4, S5 = 中指     S6, S7 = 小指
```

| 手势 | 动作说明 | S0 | S1 | S2 | S3 | S4 | S5 | S6 | S7 |
|------|---------|----|----|----|----|----|----|----|----|
| **A —** 放松 | 全部伸直 | | | | | | | | |
| **B —** 摊开 | 全部张开 | | | | | | | | |
| **C —** 握拳 | 全部弯曲 | | | | | | | | |
| **D —** 伸食指 | 仅食指伸直 | | | | | | | | |
| **E —** 伸拇指 | 仅拇指伸直 | | | | | | | | |

> 💡 **放松 (A)** 和 **摊开 (B)** 一般为全部手指伸直位，PWM 相同。
>
> **握拳 (C)** = 全部手指弯曲位。
>
> **伸食指 (D)** = 食指(S2,S3)用伸直位，其余(拇指+中指+小指)用弯曲位。
>
> **伸拇指 (E)** = 拇指(S0,S1)用伸直位，其余(食指+中指+小指)用弯曲位。

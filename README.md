# CIMC - 舵机手势控制系统

基于 **STC8G1K08** 单片机 + **PCA9685** PWM 驱动芯片，控制 **8 路舵机**实现多种手势的嵌入式项目。

每根手指由 **2 个舵机拮抗驱动**（共 4 指：拇指、食指、中指、小指）。

## 📁 项目结构

```
CIMC/
├── 51/                       # 主项目 - 串口指令手势控制
│   ├── 51.c                  # 源代码（A-E 五手势，串口中断响应）
│   ├── 51.uvproj             # Keil µVision5 项目文件
│   └── STARTUP.A51           # Keil C51 启动代码
│
├── 51_test/                  # 测试项目 - 串口交互式舵机调试
│   ├── 51.c                  # 源代码（串口指令调参，含耦合/个体模式）
│   ├── 51.uvproj             # Keil µVision5 项目文件
│   ├── STARTUP.A51           # Keil C51 启动代码
│   ├── COMMANDS.md           # 📖 完整指令手册与调试流程
│   └── COMMANDS.pdf          # 指令手册 PDF 版
│
└── README.md
```

## 🖐 手势指令（51 主项目）

通过串口发送单字符指令（**115200 baud**），MCU 回复 `X OK\r\n` 确认：

| 指令 | 手势 | 说明 |
|------|------|------|
| `A` | 放松 (Relax) | 全部手指放松 |
| `B` | 摊开 (Spread) | 全部手指张开 |
| `C` | 握拳 (Fist) | 全部手指握紧 |
| `D` | 伸食指 (Point) | 仅食指伸出 |
| `E` | 伸拇指 (ThumbUp) | 仅拇指翘起 |

> 各手势 PWM 值在 `51.c` 顶部以宏定义配置，当前使用 **230-370 范围**（较安全的中等行程）。

## 🔧 舵机对应关系

| 舵机通道 | 对应手指 | 弯曲方向 |
|---------|---------|---------|
| 0, 1 | 大拇指 (Thumb) | S0↑ S1↓ |
| 2, 3 | 食指 (Index) | S2↑ S3↓ |
| 4, 5 | 中指 (Middle) | S4↑ S5↓ |
| 6, 7 | 小指 (Pinky) | S6↑ S7↓ |

> **拮抗驱动**：每个手指对中，前一舵机增大 + 后一舵机减小 → 弯曲；反之伸直。

## 🧪 串口交互式舵机调试（51_test）

`51_test` 是一个功能完整的**串口交互式舵机调试工具**，用于精确标定每个舵机的角度范围。

### 快速开始

上电后通过串口（115200 baud）连接，MCU 会发送 3 次欢迎信息：

```
=== Servo Tuning Tool ===
115200 baud | Send 'h' for help
>
```

### 指令概览

| 指令 | 功能 |
|------|------|
| `0` ~ `7` | **选中舵机 N**，进入单舵机模式 |
| `f0` ~ `f3` | **选中手指对**（0=拇指 1=食指 2=中指 3=小指） |
| `i` / `I` | **个体模式**：在手指对内切换控制单个舵机 |
| `b` / `B` | **返回耦合模式** |
| `+` | 步进+（耦合=弯曲，单舵机/个体=PWM↑） |
| `-` | 步进-（耦合=伸直，单舵机/个体=PWM↓） |
| `s` | 打印全部 8 个舵机 PWM |
| `Sxx` | **直接设置步长**，如 `S25` → step=25 |
| `c` / `C` | 全部舵机回中（300） |
| `n` / `N` | **切换步长**（10→20→50→1→5→循环） |
| `a` / `A` | **自动扫角**：当前手指对完整扫一遍 |
| `p` / `P` | 显示当前选中信息 |
| `h` / `H` | 打印帮助列表 |

> **详细指令说明与调试流程请参见 [`COMMANDS.md`](51_test/COMMANDS.md)**。

### 三种控制模式

```
单舵机模式         手指对耦合模式        手指对个体模式
  0-7      ──→      f0-f3       ──→        i
  ± 调单个舵机     ± 同步弯曲/伸直      ± 调对中单个舵机
                       ←── b ──→
```

## 🛠 开发环境

| 项目 | 值 |
|------|-----|
| **IDE** | Keil µVision5 |
| **MCU** | STC8G1K08 |
| **PWM 驱动** | PCA9685 (I²C 地址 0x40) |
| **晶振** | 11.0592MHz |
| **串口** | 115200 baud, 8N1 (TH1=0xFD, 1T 模式) |
| **I²C 引脚** | P3.2 (SCL), P3.3 (SDA) |

## 📥 烧录步骤

1. 在 Keil 中打开 `51/51.uvproj` 或 `51_test/51.uvproj`
2. **Project → Rebuild all target files** (F7) 编译
3. 打开 AiCube-ISP（或 STC-ISP）烧录工具
4. 选择生成的 `.hex` 文件（在对应 `Objects/` 目录下）
5. 连接 STC8G1K08，点击下载/编程
